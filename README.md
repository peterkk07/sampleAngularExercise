# sampleAngularExercise

Installation
============

Clone project

```bash
$ git clone https://github.com/peterkk07/sampleAngularExercise.git
```

Go within the Angular project

```bash
$ cd sample-app
```

Npm install

```bash
$ npm install
```

Run the project

```bash
$ npm run start
```

Open your browser with the port 4200
```bash
http://localhost:4200/
```

Specifications
============

Angular version 8.1.3

Angular CLI: 8.1.3

typescript 3.4.5

node 10.9 or greater
